# 💰 Expense Tracker (Flux Finance)

A clean, modern personal finance tracker built with vanilla HTML, CSS, and JavaScript.
Track income/expenses, view a live-updating dashboard, and manage your savings goals.

🔗 **Live Demo:** [flux-finance-tracker.netlify.app](https://YOUR-LIVE-DEMO-LINK/)

![Demo](./finance%20tracker/asserts/logo.png)

## ✨ Features

- 🔐 Sign up + login + logout (simple session)
- ➕ Add income & expenses with categories
- 📊 Dashboard KPIs (total balance, last 30 days income/expense) update automatically
- 🧾 Transactions list with delete, export JSON, import JSON, clear all
- 🎯 Goals: create goals, view goals list, delete, export/import JSON, clear all
- 🔎 Quick search filter on dashboard transactions
- 📱 Responsive UI (dashboard layout + panels)

## 🛠️ Tech Stack

- HTML5, CSS3
- JavaScript (Vanilla)
- Browser Storage:
  - `localStorage` (users / login session)
  - `sessionStorage` (transactions + goals)
